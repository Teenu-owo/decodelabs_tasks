Data Analytics Intern - Decodelabs

This repository showcases the projects completed during my 4-week Data Analytics Internship at DecodeLabs, covering data cleaning of an e-commerce dataset, descriptive statistical analysis, SQL querying, and the development of an interactive Excel dashboard to generate business insights.
## Project Structure
Week 1: Data Cleaning
* Objective: Clean and prepare raw e-commerce data for analysis.
* Tools Used: Power Query and Excel.
* What i did: Cleaned the dataset by replacing misiing coupon codes with "No Coupon", removing duplicates, and standardizing the date and currency formats.
* Folder Location: 'Week 1- Data Cleaning'

Week 2: Exploratory Data Analysis (EDA)
* Objective: Ananlyze the e-commerce dataset to identify patterns, trends, outliers and business insights.
* Tools Used: Excel (Pivot Tables, Descriptive Statistics Formulas).
* What i did: Performed descriptive statistics, identified outliers using the IQR method, created PivotTables, and summarized key business insights.
* Folder Location: 'Week 2- Exploratory Data Analysis'

Week 3: SQL Data Analysis
* Objective: Use SQL queries to extract meaningful insights from the e-commerce dataset.
* Tools Used: SQL Server / SQL Server Management Studio (SSMS).
* What i did: Wrote SQL queries using SELECT, WHERE, ORDER BY, GROUP BY, COUNT(), SUM(), and AVG() to analyze sales performance, customer behavior, and product trends.
* Folder Location: 'Week 3- SQL Data Analysis'
